To reproduce results from Assign2.py simply run script python3 Assign2.py

Note: you must have libraries such as numpy pandas matplotlib, to download them use command 'pip install numpy' in terminal and so on.

Then you can simply run the script, or open it in development environment such as Pycharm and run it.

The script will first output one plot, after closing it, it will output second plot, and after closing it, the script will output every
calculation regarding logistic regression.

Important part of codes are commented.
